# azcam-vatt4k

## Purpose

This repository contains the *azcam-vatt4k* *azcam* environment. It contains code and data files for the VATT Vatt4k camera system maintained by the UArizona Imaging Technology Laboratory.

## Installation Example

Azcam code is typically run directly from the installation folders to allow changes. From `c:\azam` folder...

```
git clone https://github.com/uaitl/azcam-vatt4k
pip install -r azcam-vatt4k\requirements.txt
```

For Windows systems the normal destination is `c:\azcam\azcam-vatt4k`.
